# Bakomon (Baca Komik Online)

Bakomon is simple tool which allows you to scrap manga/manhwa/manhua from online comic readers.

Currently supported sites are:
- bacakomik.co
- and more.

## [Demo](https://bakomon.blogspot.com/?demo)
